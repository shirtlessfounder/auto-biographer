# auto-biographer

Small TypeScript service foundation for auto-biographer, the semiautonomous X posting workflow.

## Local setup

```bash
cd /Users/dylanvu/auto-biographer
pnpm install
```

Copy `.env.example` into your local environment before running commands.

## Commands

```bash
pnpm test
pnpm typecheck
pnpm cli -- migrate
pnpm cli -- check-env
```
